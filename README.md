# arduino-in-rust-led
===================

Проект Rust для _Arduino Mega 2560

## Інструкції зі збірки 
1. Встановіть необхідні компоненти, як описано у файлі README [`avr-hal` (`avr-gcc`, `avr-libc`, `avrdude`, [`ravedude`]).
2. Запустіть `cargo build` для збірки прошивки.
3. Запустіть `cargo run` для прошивки підключеної плати. Якщо `ravedude` не виявляє вашу плату, перевірте її документацію за адресою
4. `ravedude` відкриє консольний сеанс після прошивки, де ви зможете взаємодіяти з консоллю UART вашої плати.
5. 
[`avr-hal` README]: https://github.com/Rahix/avr-hal#readme
[`ravedude`]: https://crates.io/crates/ravedude

# Про кнопку
Термін «кнопка» має кілька основних значень залежно від сфери застосування: від дрібної канцелярії до складних елементів комп'ютерних інтерфейсів.
Основні значення
-В техніці та електроніці: Це пристрій, призначений для замикання або розмикання електричного кола шляхом натискання. Кнопки використовуються в побутовій техніці, клавіатурах, пультах керування тощо. Спеціалізовані варіанти включають тривожні кнопки для виклику охорони або кнопки пожежної сигналізації.
-В інформатиці (UI/UX): Це елемент графічного інтерфейсу користувача, який при натисканні (кліку) ініціює певну дію або подію в програмі чи на вебсайті. Прикладом є меню «Пуск» у Windows або кнопки дій у презентаціях.
-В одязі: Це металева або пластикова застібка, що складається з двох частин, які замикаються одна в одну при натисканні. Популярними є типи «Альфа» або «Каппа».
-Канцелярська кнопка: Короткий металевий гвіздок із широкою головкою, що використовується для прикріплення паперу до м'яких поверхонь (наприклад, до коркової дошки).

# Про світлодіод
Світлодіод (LED — light-emitting diode) — це напівпровідниковий прилад, який перетворює електричний струм безпосередньо на світлове випромінювання (електролюмінесценція). Це ефективні, довговічні джерела світла, що майже не нагріваються, використовуються для освітлення та індикації. Світлодіоди випромінюють світло вузького спектра, колір якого залежить від хімічного складу матеріалу.
Основні характеристики світлодіодів:
-Принцип дії: світяться, коли струм проходить через -перехід (від анода до катода).
-Типи: індикаторні (низька потужність) та освітлювальні (висока яскравість).
-Переваги: висока енергоефективність, довговічність (до 10 років), миттєве вмикання, відсутність ртуті.
-Застосування: побутове освітлення, автомобільні фари, індикатори на приладах, екрани, вуличне освітлення.

# Про Arduino
Arduino (Ардуіно) — апаратна обчислювальна платформа для аматорського конструювання, основними компонентами якої є плата мікроконтролера з елементами вводу/виводу та середовище розробки Processing/Wiring на мові програмування, що є спрощеною підмножиною C/C++. Arduino може використовуватися як для створення автономних інтерактивних об'єктів, так і підключатися до програмного забезпечення, яке виконується на комп'ютері (наприклад: Processing, Adobe Flash, Max/MSP, Pure Data, SuperCollider). Інформація про плату (малюнок друкованої плати, специфікації елементів, програмне забезпечення) знаходяться у відкритому доступі і можуть бути використані тими, хто воліє створювати плати власноруч.
Назва Arduino походить від бару в Івреа, Італія, де зустрічалися деякі із засновників проекту. Бар був названий на честь Ардуїн I, який був маркграфом Маршу Івреї та королем Італії з 1002 по 1014 роки.
# Про Arduino Mega 2560
Arduino Mega 2560 — це потужна мікроконтролерна плата на базі ATmega2560, розроблена для складних проектів, таких як 3D-принтери, робототехніка та розумні будинки. Вона має 54 цифрові входи/виходи (14 ШІМ), 16 аналогових входів, 256 КБ флеш-пам'яті та працює на частоті 16 МГц, пропонуючи значно більше ресурсів, ніж Arduino Uno. 
Основні характеристики та особливості:
-Мікроконтролер: ATmega2560 (16 МГц).
-Пам'ять: 256 КБ флеш-пам'яті, 8 КБ SRAM, 4 КБ EEPROM.
-Інтерфейси: 4 апаратних UART (послідовних порти), що забезпечує гнучкість у підключенні.
-Живлення: Робоча напруга 5В, рекомендована вхідна напруга 7-12В.
-Сумісність: Розроблена для сумісності з більшістю шилдів (плат розширення) для Arduino Uno.
-Плата є чудовим вибором, коли бракує портів або пам'яті на простіших моделях, таких як Arduino Uno.
